TTL

Description: Returns the remaining time to live of a key that has a timeout.

Return values:

  • -2 if the key does not exist.
  • -1 if the key exists but has no expiration.
  • >= 0 remaining time to live in seconds.
> SET mykey hello
+OK
> EXPIRE mykey 10
:1
> TTL mykey
:9
